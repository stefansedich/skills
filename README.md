# Skills

A collection of personal [GitHub Copilot skills](https://docs.github.com/en/copilot).

## Skills

- [leet](skills/leet/SKILL.md) — Respond entirely in leet speak (1337 5p34k).

## Installation

Install with the [`gh skill`](https://cli.github.com/manual/gh_skill) CLI:

```sh
# Install all skills from this repo
gh skill install stefansedich/skills

# Or install an individual skill
gh skill install stefansedich/skills <skill>
```
